Colonel General — Quick Start

This is a dual-purpose platform: a working AI wargaming persona and an interactive prompt engineering course.

Getting Started

  1. Choose a provider and model from the dropdowns above
  2. Enter your API key (see links below)
  3. Click Connect
  4. Present a scenario and start the wargame

Get an API Key (Free Options Available)

  • Google Gemini — FREE tier, great starting point
  • OpenRouter — access to many models, some FREE (DeepSeek R1, Qwen)
  • Anthropic — best persona consistency, requires paid account

Learning Features

  • System Prompt tab — view the full prompt with toggle switches to enable/disable sections and see how it changes AI behavior
  • Edit Mode — modify the prompt live and chat with your modified version
  • Guide tab — three-perspective analysis of every section (Doctrinal Accuracy, Officer Realism, Prompt Engineering)
  • Sources tab — full bibliography of Russian military doctrine and prompt engineering research

Wargame Modes

  • Freeplay — open-ended. Present any scenario.
  • Scripted — provide predetermined actions for analysis
  • Seminar — analytical discussion and teaching mode
  • Individual Move — quick turn-based decisions

Privacy

Your API key is stored only in your browser's localStorage. It is never sent anywhere except directly to your chosen AI provider's API. No analytics, no tracking, no backend.
